# 📊 路徑圖表使用指南

## 🎨 Excel 路徑視覺化

GPS 油單輔助系統現在會自動生成三個工作表，其中「路徑圖表」工作表可以讓你在 Excel 中繪製路徑地圖！

---

## 📂 三個工作表

匯出的 Excel 包含：

### 📊 工作表 1：行程記錄
```
用途：行程摘要
內容：
• 行程名稱
• 起終點
• 時間
• 里程數
• 車輛資訊
```

### 🗺️ 工作表 2：路徑座標
```
用途：詳細座標清單
內容：
• 每個 GPS 點的座標
• Google Maps 連結
• 可驗證路徑
```

### 📈 工作表 3：路徑圖表 ⭐ 新增！
```
用途：繪製路徑地圖
內容：
• 行程名稱
• 經度（X軸）
• 緯度（Y軸）
• 點序號

特色：
✅ 專為 Excel 散點圖設計
✅ 可視覺化路徑
✅ 支援多行程比較
✅ 自動分組
```

---

## 🎯 如何繪製路徑圖

### 方法 1：快速建立（推薦）

**步驟：**

1. **開啟 Excel 檔案**
   ```
   雙擊下載的 Excel
   ```

2. **切換到「路徑圖表」工作表**
   ```
   點擊底部分頁標籤
   ```

3. **選取資料範圍**
   ```
   選取 B 欄（經度）和 C 欄（緯度）
   
   範例：
   • 單一行程：B2:C200
   • 多個行程：B2:C1000
   • 或直接選取整個 B 和 C 欄
   ```

4. **插入散點圖**
   ```
   插入 → 圖表 → 散點圖 → 帶平滑線的散點圖
   ```

5. **完成！**
   ```
   看到路徑地圖了！
   ```

---

### 方法 2：分別繪製各行程（多行程）

如果要讓不同行程用不同顏色顯示：

**步驟：**

1. **篩選第一個行程**
   ```
   資料 → 篩選
   點擊 A 欄標題的下拉箭頭
   只勾選第一個行程名稱
   ```

2. **選取並插入圖表**
   ```
   選取 B 欄和 C 欄（篩選後的資料）
   插入 → 散點圖 → 帶平滑線
   ```

3. **新增其他行程**
   ```
   右鍵圖表 → 選取資料
   → 新增系列
   → 選取其他行程的 B 欄和 C 欄
   ```

4. **設定顏色**
   ```
   點擊每條線
   → 設定格式
   → 線條顏色
   → 選擇不同顏色
   ```

## 🎨 美化圖表

### 基本設定

**圖表標題：**
```
點擊標題 → 輸入
例如：「GPS 行程路徑圖」
```

**座標軸標題：**
```
圖表工具 → 設計 → 新增圖表元素 → 座標軸標題

X 軸：經度
Y 軸：緯度
```

**圖例：**
```
顯示行程名稱
位置：右側或底部
```

---

### 進階美化

**調整線條：**
```
右鍵線條 → 設定資料數列格式

粗細：2-3pt
顏色：
• 行程 1：藍色
• 行程 2：綠色
• 行程 3：紅色

平滑度：0.5-1.0
```

**標記點：**
```
標記選項：
• 起點：大圓形、綠色
• 終點：大方形、紅色
• 中間點：小圓形或不顯示

大小：5-8
```

**背景：**
```
圖表區域格式：
• 填滿：淺灰色或白色
• 框線：無或淺灰
```

## 💡 實用技巧

### 技巧 1：放大特定區域
```
方法：
1. 點擊圖表
2. 右鍵 Y 軸（緯度）
3. 座標軸格式
4. 設定最小值和最大值

例如：
• 最小值：24.9
• 最大值：25.1
→ 只顯示這個範圍
```

### 技巧 2：顯示起終點
```
方法 1：標記
• 選取第一個點
• 設定為大圓形、綠色
• 選取最後一個點
• 設定為大方形、紅色

方法 2：文字標籤
• 右鍵資料點
• 新增資料標籤
• 編輯為「起點」「終點」
```

### 技巧 3：比較去回路線
```
用途：
看去程和回程是否走同一條路

方法：
1. 去程用藍色線
2. 回程用紅色線
3. 疊在一起比較
4. 看是否重疊
```

### 技巧 4：匯出圖片
```
步驟：
1. 右鍵圖表
2. 另存為圖片
3. 選擇 PNG 或 JPG
4. 可用於報告、簡報
```

## 🔧 常見問題

### Q1: 圖表看起來很奇怪？

**A: 檢查座標軸**
```
問題：X 軸和 Y 軸可能對調了

解決：
1. 右鍵圖表 → 選取資料
2. 確認：
   • X 值 = 經度（B 欄）
   • Y 值 = 緯度（C 欄）
3. 重新設定
```

### Q2: 多個行程混在一起？

**A: 分別新增系列**
```
方法：
1. 先建立第一個行程的圖表
2. 右鍵 → 選取資料 → 新增
3. 分別加入其他行程
4. 每個系列設定不同顏色
```

### Q3: 點太密集看不清楚？

**A: 調整顯示**
```
方法 1：移除標記
• 標記選項 → 無

方法 2：只顯示部分點
• 每 10 個點顯示一個標記

方法 3：放大圖表
• 拖曳圖表邊角放大
```

### Q4: 想要顯示方向？

**A: 加入箭頭**
```
方法：
1. 插入 → 圖案 → 箭頭
2. 沿著路線畫箭頭
3. 指向行進方向

或：
• 用漸層顏色
• 起點深色 → 終點淺色
```

## 📱 進階應用

### 應用 1：路線對比

**場景：比較不同日期的通勤路線**

```
步驟：
1. 匯出多天的資料
2. 在圖表中加入多條路線
3. 設定不同顏色：
```

```
展示：
1. 建立路徑圖
2. 標記 5 個客戶位置
3. 顯示行進順序

5. 附上 Excel 報告
```

### 應用 3：區域分析

**場景：分析業務範圍**

```
用途：
1. 匯出一個月的行程

```
