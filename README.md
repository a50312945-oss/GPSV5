# 🚀 快速開始

## 📱 3 步驟立即使用

### 步驟 1️⃣：開啟檔案
```
雙擊 GPS油單輔助系統_v2.0.html
或
拖曳到瀏覽器視窗
```

### 步驟 2️⃣：允許權限
```
瀏覽器詢問：「允許使用您的位置？」
→ 點擊「允許」
```

### 步驟 3️⃣：開始記錄
```
1. 點擊「開始」按鈕
2. 輸入起點（例如：公司）
3. 輸入終點（例如：客戶A）
4. 完成！開始記錄
```
